如何评论 / 讨论

每篇文章底部都有评论入口，点击会跳转到 GitHub Issues 对应页面。

评论流程

1
在文章底部点击「去 GitHub 发表讨论」按钮
2
用 GitHub 账号登录（如没有请注册，免费）
3
填写标题和内容，提交即可

💡 也可以直接打开 GitHub Issues 查看所有讨论，或新建 Issue。
