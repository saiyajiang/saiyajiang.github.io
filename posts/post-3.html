A 股量化入门：用 Python 获取和分析股票数据

2026年4月10日 约 10 分钟阅读
#Python#量化投资#金融

介绍如何用 Python 获取 A 股数据，进行基础的技术指标计算，以及可视化价格走势。适合有编程基础但对量化陌生的朋友。

数据获取

推荐使用 akshare 库，它是目前最全的中文金融数据包，免费无需注册。

pip install akshare
import akshare as ak

# 获取日线数据
df = ak.stock_zh_a_hist(symbol="000001", period="daily",
                         start_date="20230101", end_date="20261231")
print(df.head())

技术指标计算

移动平均线（MA）

df['MA5'] = df['收盘'].rolling(5).mean()
df['MA20'] = df['收盘'].rolling(20).mean()
df['MA60'] = df['收盘'].rolling(60).mean()

MACD

ema12 = df['收盘'].ewm(span=12).mean()
ema26 = df['收盘'].ewm(span=26).mean()
df['DIF'] = ema12 - ema26
df['DEA'] = df['DIF'].ewm(span=9).mean()
df['MACD'] = (df['DIF'] - df['DEA']) * 2

可视化

import matplotlib.pyplot as plt
import matplotlib.dates as mdates

plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

fig, ax = plt.subplots(figsize=(14, 7))
ax.plot(df['日期'], df['收盘'], label='收盘价', linewidth=1.5)
ax.plot(df['日期'], df['MA20'], label='MA20', alpha=0.8)
ax.plot(df['日期'], df['MA60'], label='MA60', alpha=0.8)
ax.set_title('平安银行(000001) 股价走势')
ax.legend()
plt.show()
