Dota 2 游廊地图入门
从零开始制作自定义地图

2026年4月15日 约 12 分钟阅读
#Dota2 #游戏开发 #Lua

Dota 2 的游廊（Arcade）系统允许任何人制作并发布自定义游戏地图。本文介绍如何从零开始入门地图制作。

环境准备

安装 Dota 2 Workshop Tools

在 Steam 的 Dota 2 属性页面中，找到 DLC 标签，勾选安装 Dota 2 Workshop Tools DLC。安装完成后，在游戏启动选项中选择"进入地图编辑器"即可打开工具套件。

工具结构

Workshop Tools 包含以下核心组件：

  • Hammer：地图地形编辑器
  • Asset Browser：资源浏览器
  • Panorama UI：自定义 HUD 界面（HTML/CSS/JS）
  • KV 编辑器：物品、技能等游戏数据

Lua 脚本入门

游廊地图的游戏逻辑完全由 Lua 脚本驱动。入口文件是 vscripts/addon_game_mode.lua。

function Activate()
  GameRules:GetGameModeEntity():SetThink("OnThink", GameRules:GetGameModeEntity(), 1)
  ListenToGameEvent("game_rules_state_change", OnStateChange, nil)
end

function OnStateChange()
  local state = GameRules:State_Get()
  if state == DOTA_GAMERULES_STATE_GAME_IN_PROGRESS then
    print("游戏开始！")
  end
end

常用游戏事件

  • npc_spawned — 单位生成
  • entity_killed — 单位死亡
  • dota_player_gained_level — 玩家升级
学习地图制作最快的方式，是找一个已有的开源地图，把代码读懂，然后改它。
